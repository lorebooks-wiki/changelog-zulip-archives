For the first time, I printed an item that needed a colour-change partway through, and it worked!

It was a bit of a confusing process, as the printer starts ejecting the current filament but continues to print in a sort of junk area, then it pauses, then the filament should be swapped, then it continues printing in the junk area for a bit, and then it resumes printing the desired object

But, I didn't know the GCODE would include those pause and swap instructions, so I manually paused it, swapped, and resumed, which ended up being quite redundant

But now I know for next time :)
